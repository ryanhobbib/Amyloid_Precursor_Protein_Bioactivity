# Reproducing this web app
To recreate this web app on your own computer, do the following.

### Create conda environment
Create a conda environment called *bioactivity*
```
conda create -n bioactivity python=3.7.9
```
Activate the *bioactivity* environment
```
conda activate bioactivity
```
### Install prerequisite libraries

Download requirements.txt file

```
wget https://github.com/ryanhobbib/Amyloid_Precursor_Protein_Bioactivity/blob/main/bioactivity-prediction-app-main/requirements.txt

```

Pip install libraries
```
pip install -r requirements.txt
```

###  Download and unzip contents from GitHub repo

Download and unzip contents from https://github.com/ryanhobbib/Amyloid_Precursor_Protein_Bioactivity/blob/main/bioactivity-prediction-app-main.zip

### Generating the PKL file

The machine learning model used in this web app will firstly have to be generated by successfully running the included Jupyter notebook [bioactivity_prediction_app.ipynb](https://github.com/ryanhobbib/Amyloid_Precursor_Protein_Bioactivity/blob/main/bioactivity-prediction-app-main/bioactivity_prediction_app.ipynb). Upon successfully running all code cells, a pickled model called amyloid_beta_model.pkl will be generated.

###  Launch the app

```
streamlit run app.py
```
