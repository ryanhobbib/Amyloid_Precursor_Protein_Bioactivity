# Amyloid_Precursor_Protein_Bioactivity
My first bioinformatics project.
This project was completed following the [bioinformatics course by Chanin Nantasenamat (aka Data Professor)](https://www.youtube.com/watch?v=jBlTQjcKuaY) with modifications to investigate the Beta-Amyloid Precursor Protein target.

The goal of this project was to build an interactive tool where users can input a molecular structure (SMILES string) of their choice and determine the IC50 value of that molecule on the Amyloid-Beta Precursor Protein. This protein is one of the main proteins involved in the progression of Alzheimer's disease, and this tool could provide a way for researchers to investigate the bioactivity of a drug on this protein.

### File Structure
#### Main
The main folder includes all of the steps involved in the data analysis pipeline, including EDA, data processing, model building, and model comparison steps.

#### data
The data folder includes all datasets involved in this pipeline, from the raw data to the output data displayed in the interactive tool.

#### results
The results folder includes all figures and plots from EDA.

#### bioactivity-prediction-app-main
The bioactivity-prediction-app-main folder contains all files necessary for running the prediction app.
